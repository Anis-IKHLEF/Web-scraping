#   Web Scraping Project: Company Data Extraction for Job Search

##   Project Overview
This project extracts company data from the Kompass directory for the **Bejaia region in Algeria** to create a targeted list of potential employers for job applications. The solution overcomes website anti-scraping measures using professional tools and produces a structured Excel database of local businesses.

##   Features
- **410+ companies** extracted with detailed information
- **Bypasses 403 Forbidden errors** using Bright Data API
- **Multi-page pagination handling** for complete data collection
- **Structured Excel output** for easy job search management
- **Clean data extraction** including company names, locations, and phone numbers

##  Target Industries Identified
- Manufacturing & Industry
- Healthcare & Medical Services
- Agriculture & Food Processing
- Construction & Engineering
- Technology & IT Services
- Tourism & Hospitality

##  Technologies Used
- **Python 3.13.5**
- **BeautifulSoup4** - HTML parsing
- **Pandas** - Data manipulation and Excel export
- **Requests** - HTTP requests
- **Bright Data API** - Professional web scraping service
